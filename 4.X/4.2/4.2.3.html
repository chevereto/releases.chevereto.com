# Chevereto 4.2.3

2024-11-18

  • Added Bulk importer as a core feature (opens new window)
  • Added translation placeholder for "hosted at" titles (opens new window)
  • Added Akismet spam protection on tags (opens new window)
  • Improved install screen (opens new window)
  • Changed GMT references to UTC (opens new window)
  • Updated Polish, Portuguese, Portuguese (Brazil) and Vietnamese translations (opens new window)
  • Fixed bug with duplicated tags display (opens new window)
  • Fixed bug with missing SQL update affecting Bulk importer (opens new window)
  • Fixed bug with fail-safe ENV using array values (opens new window)
  • Fixed bug with wrong column type for image_original_exifdata (opens new window)
  • Fixed bug with Bulk importer not skipping missing directories (opens new window)
  • Fixed bug with Open Graph double encoding (opens new window)
  • Fixed bug with email subject extra encoding (opens new window)
  • Fixed bug with Disqus dark theme detection/refresh (opens new window)
  • Fixed bug with not working cookie law acceptance (opens new window)
  • Fixed bug with tag filtering using /?match=all (opens new window)
  • Fixed bug with PUP.js auto-close behavior (opens new window)
